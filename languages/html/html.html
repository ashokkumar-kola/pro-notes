MULTIMEDIA

<marquee> MARQUEE TAG </marquee> marquee

  • behavior = "scroll | slide | alternate" (Defines scrolling behavior)
  • direction = "left | right | up | down" (Scrolling direction)
  • scrollamount = "number" (Speed of movement)
  • loop = "infinite | number" (Number of loops)
  • bgcolor = "color" (Background color)
  • hspace = "number" | vspace = "number" (Horizontal/Vertical spacing)

<img> IMAGE TAG

  • src = "url" (Image source URL)
  • alt = "text" (Alternative text for accessibility)
  • width = "number" | height = "number" (Dimensions)
  • loading = "eager | lazy" (Loading strategy)
  • decoding = "sync | async | auto" (Decoding method)
  • crossorigin = "anonymous | use-credentials" (CORS policy)
  • referrerpolicy = "no-referrer | origin | strict-origin-when-cross-origin" (Referrer policy)

<audio> AUDIO TAG </audio>

  • src = "url" (Audio file URL)
  • controls (Shows audio controls)
  • autoplay (Plays automatically)
  • loop (Repeats audio)
  • muted (Starts muted)
  • preload = "none | metadata | auto" (Loading behavior)
  • crossorigin = "anonymous | use-credentials" (CORS policy)

<video> VIDEO TAG </video>

<video> <source> </video>

  • src = "url" (Video file URL)
  • width = "number" | height = "number" (Dimensions)
  • poster = "url" (Thumbnail image before playing)
  • controls (Shows playback controls)
  • autoplay (Starts playing automatically)
  • loop (Repeats video)
  • muted (Starts muted)
  • preload = "none | metadata | auto" (Preload behavior)
  • playsinline (Prevents fullscreen mode on mobile)
  • crossorigin = "anonymous | use-credentials" (CORS policy)
  • referrerpolicy = "no-referrer | origin | strict-origin-when-cross-origin" (Referrer policy)
